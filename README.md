# ⚡ AutoML Pipeline Builder

> 🚧 Work in progress

An automated ML pipeline that takes any CSV and returns a trained model, metrics, SHAP analysis and PDF report — no coding required.

## Features (in progress)
- 🤖 Auto-detects classification vs regression
- 📊 Trains and compares 4 models (Logistic Regression, Random Forest, XGBoost, LightGBM)
- 🔍 SHAP feature importance explanation
- 📥 Downloadable trained model (.pkl)
- 📄 Downloadable PDF report

## Tech Stack
Python · Streamlit · Scikit-learn · XGBoost · LightGBM · SHAP · ReportLab
